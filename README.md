# Data-Science-IIITB

This repository contains resources, code, and materials related to Data Science coursework and projects at IIIT Bangalore, implemented using the R programming language.

## Overview

- **Course/Project:** Data Science (IIIT Bangalore)
- **Primary Language:** R
- **Repo Description:** R

## Contents

You can expect to find:

- R scripts for various data science concepts and assignments
- Example datasets for practice and demonstration
- Notes and documentation (if provided)
- Project work and solutions

## Requirements

- [R](https://www.r-project.org/) (100% of the codebase)
- Recommended: [RStudio](https://posit.co/products/open-source/rstudio/) for ease of use

## Getting Started

Clone the repository:

```bash
git clone https://github.com/nitinsriv/Data-Science-IIITB.git
cd Data-Science-IIITB
```

Open the `.R` scripts in RStudio or run them via command line:

```r
source("filename.R")
```

## Usage

Feel free to explore the scripts, run the analyses, and use or modify the datasets provided for your learning or coursework purposes.
